Painel de saldos separado (resumo rápido)

Uma seção colapsável no topo do dashboard mostra todos os saldos de uma vez — sem poluir os cards. Ideal para a rotina de "checar saldos antes de começar o dia".

Os dois: painel resumo + saldo no card

Painel colapsável no topo para visão geral + saldo visível em cada card individualmente. Mais informação de uma vez, mais espaço ocupado.

O saldo é colorido: 🟢 saudável · 🟡 atenção (abaixo de R$500) · 🔴 crítico (abaixo de R$200).
